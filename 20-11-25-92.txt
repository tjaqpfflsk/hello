1.키워드란 무엇인가?

   이런 단어들은 변수명으로 사용할 수 없다.
--------------------
   키워드란 미리 예약해 놓은 단어를 의미합니다.
   그래서 키워드를 Reserved Word라고 합니다.
   결국 키워드는 특별한 의미를 갖고 있는 명령어입니다.

2.2의 보수법이란?
   2의 보수법이란 정수를 컴퓨터가 알 수 있는 음수로 만들 때 하는 방식이다.
   일단 정수를 2진수로 바꾼 다음에 1의 보수로 바꿔준 뒤 1을 더하면 된다.
   이를 2의 보수법이라고 한다.
--------------------
   'n의 보수'는  어떤 수에 더했을 때 n이 되는 수이다.
   
3.컴퓨터에서 음의 정수 표현방법에 대하여 설명하시오.
   부호비트를 주어 0은 + 이고 1은 -로 하여 맨 앞자리 수를 1로 바꾸는 방법이 있으나
   실질적으로 계산을 할 때 00000000 로 떨어지지 않는 일이 발생하여
   2의 보수법을 진행하여 음의 정수로 표현이 된다.

4.-5가 컴퓨터에서의 2진수 값을 나타내시오.
   일단 5를 2진수로 표현하게 되면 00000101
   여기에 1의 보수를 해주면         11111010
   여기에 1을 더해주면                11111011
   결과 컴퓨터에서 -5 값은 11111011 이다.

5.실수에서 오차가 생기는 원인은 무엇인가?
   실수 어느 두 수 사이의 실수는 무한으로 존재하며 그래서 실수의 범위를 정해 놓았다
   범위 안의 실수도 무한하기 때문에 실수로 사용하는 값들을 정해놓았다.
   그래서 우리가 지정한 실수 값이 그대로 사용되는 것이 아니라 지정한 수들 중 
   그 값의 가장 근사한 값으로 연산이 되기 때문이다.
-------------------
   지수부와 가수부를 사용하면서 표현할 수 있는 범위는 늘어나지만, 
   10진수를 정확하게 표현할 수는 없게 됩니다.
   따라서 컴퓨터에서 실수를 표현하는 방법은 정확한 표현이 아닌 언제나 근사치를 표현할 뿐임을
   항상 명심해야 한다.

6.실수의 표현법은(고정소수법, 부동소수법)을 설명하시오.
   고정소수법은 이미 숫자에서 소수점 자리가 고정된 것을 의미하며 
   예를 들 경우 3.14 나 0.928 같은 경우이다.
   부정소수법은 소수점이 이동이 가능한 것을 말하며
   예를 들 경우 3.14 x 10 또는 0.92823 x e3 을 말한다.
--------------------
   - 고정소수법 :    
      실수는 보통 정수부와 소수부로 나눌 수 있습니다.
      따라서 실수를 표현하는 가장 간단한 방식은 소수부의 자릿수를 미리 정하여
      고정된 자릿수의 소수를 표현하는 것입니다.
   - 부동소수법 : 
      실수는 보통 정수부와 소수부로 나누지만, 가수부와 지수부로 나누어 표현할 수도 있습니다.
      부동 소수점 방식은 이렇게 하나의 실수를 가수부와 지수부로 나누어 표현하는 방식입니다.

      고정 소수점 방식은 제한된 자릿수로 인해 표현할 수 있는 범위가 매우 작습니다.
      하지만 부동 소수점 방식은 다음 수식을 이용하여 매우 큰 실수까지도 표현할 수 있습니다.

7.정밀도란 무엇인가요?
   실수에서 얼마나 오차범위가 작은지를 말하며 
   float 의 경우 소수점 아래 6자리 정밀도
   double의 경우 소수점 아래 15자리 정밀도를 가진다.
   하지만 이 정밀도는 연산하기 전이며 연산하면 정밀도는 더 떨어진다.

8.상수란 무엇이며 변수와의 차이는?
   상수는 고정된 값을 말하며 한번 초기화를 시킨 경우 그 값은 변할 수 없다.
   변수는 변할 수 있는 수를 말하며 초기화를 한 경우에도 그 값은 변할 수 있다.
   상수는 보통 고정적으로 누구가 바꿀 수 없도록 하기 위해서 사용한다. 
   이미 정해져 있는 값을 사용할 때인데 예로는 final double PI = 3.14; 

9.초기화란 무엇인가?
   변수선언을 할 때 처음으로 값을 줄 때를 초기화라고 한다. 
   변수 선언과 초기화는 분리하여 할 수 있다.
   예)  int a;
         a = 1;
         같이 하는 경우 int b = 2;
 
10.리터럴이란 무엇인가요?
   우리가 변수를 선언할 때 주는 값을 리터럴이라고 한다. 
   int a = 12345; 라고 할 때 매번 숫자로 부르지 않고 간단히 부르기 위함이다.
 
11.아래의 내용이 에러가 나는 이유와 수정방법은?
System.out.println(3147483647 + 3147483648);

   자바시스템에서는 정수를 기본적으로 int 타입으로 잡는데 int의 범위는 21억이다.
   그런데 int 값의 범위를 넘었기 때문에 long 타입을 사용해야 한다.
   이럴 때에는 리터럴 뒤에 대문자L나 소문자l을 넣어주면 해결된다. 
System.out.println(3147483647L + 3147483648L);

12.자동 형변환과 명시적 형변환에 대하여 설명하시오.
   자동 형 변환 : cpu는 연산을 할 때 한 가지 타입끼리만 연산을 할 수 있다.
	        그래서 연산을 할 때 타입이 서로 다른 경우 한 가지 타입으로 맞추어 주는데
                      이를 자동형변환이라고 한다. 
	        크기가 더 큰 타입으로 변환한다. 
                      실수와 정수의 연산에 경우 소수점 아래를 남기기 위해서 실수타입으로 변환한다.
   명시적 형 변환 : 명시적 형 변환은  자동형 변환과는 반대로 큰 크기의 타입에서 작은 크기의 타입으로 변환하거나
                         실수에서 정수 타입으로 변화하는 경우에 강제적으로 형을 변환시키는 것을 말한다.

13.아래의 에러가 나는 이유를 설명하고 수정하시오.
    	int a = 3;
    	int b = 4;
    	
    	double result = a / b;
    	
    	System.out.println(result);

   연산을 할 때 인트 나누기 인트를 해서 결과값이 인트이므로 결과 값인 인트 0을 더블 타입으로
   출력한 것이므로 0.0이 나오는 것이다. 이 경우 int a, b를 double로 바꾸거나 result 값에서 
   a,b를 double로 바꾸면 된다.  

	int a = 3;
    	double b = 4;
    	
    	double result = a / b;
    	
    	System.out.println(result);
________________________________
	int a = 3;
    	int b = 4;
    	
    	double result = (double)a / b;
    	
    	System.out.println(result);
